# LinkFlip - デザインシステム

## 1. デザイン哲学

**Glassmorphism UI** と **温かみのあるグラデーション** を組み合わせたモダンで親しみやすいデザイン。

- **Glassmorphism**: 透明性、ぼかし効果、階層感を表現
- **温かみのあるグラデーション**: オレンジ、ピーチ、ゴールドなどの暖色系で親しみやすさを演出
- **親しみやすさ**: ビジネス利用でも日常利用でも使いやすい雰囲気

## 3. Glassmorphism エフェクト

### 3.1 ガラスカード（SNSカード）

**構成要素**:
- **背景**: `rgba(255, 255, 255, 0.8)`（ライト）/ `rgba(30, 28, 24, 0.8)`（ダーク）
- **ぼかし効果**: `backdrop-filter: blur(10px)`
- **ボーダー**: `1px solid rgba(255, 255, 255, 0.2)`（ライト）/ `1px solid rgba(255, 255, 255, 0.1)`（ダーク）
- **角丸**: `16px`
- **シャドウ**: `0 8px 32px rgba(0, 0, 0, 0.1)`（ライト）/ `0 8px 32px rgba(0, 0, 0, 0.3)`（ダーク）

**CSS例**:
```css
.glass-card {
  background: rgba(255, 255, 255, 0.8);
  backdrop-filter: blur(10px);
  border: 1px solid rgba(255, 255, 255, 0.2);
  border-radius: 16px;
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.1);
}
```

### 3.2 ガラスボタン

**構成要素**:
- **背景**: グラデーション + `backdrop-filter: blur(8px)`
- **ぼかし効果**: `blur(8px)`
- **ボーダー**: `1px solid rgba(255, 255, 255, 0.3)`
- **角丸**: `12px`
- **パディング**: `12px 24px`

**CSS例**:
```css
.glass-button {
  background: linear-gradient(135deg, rgba(255, 154, 86, 0.8), rgba(255, 179, 71, 0.8));
  backdrop-filter: blur(8px);
  border: 1px solid rgba(255, 255, 255, 0.3);
  border-radius: 12px;
  padding: 12px 24px;
}
```

### 3.3 ガラスモーダル

**構成要素**:
- **背景**: `rgba(0, 0, 0, 0.4)` + `backdrop-filter: blur(4px)`
- **コンテンツ背景**: ガラスカードと同じ
- **ぼかし効果**: `blur(4px)`
- **角丸**: `20px`

## 9. アニメーション

### 9.1 Flip アニメーション

**仕様**:
- **継続時間**: 300ms
- **イージング**: `ease-out`
- **変形**: Y軸回転（0° → 180°）

### 9.2 プレス状態アニメーション

**カード**:
- **スケール**: 0.95
- **不透明度**: 0.8
- **継続時間**: 150ms

**ボタン**:
- **スケール**: 0.97
- **継続時間**: 100ms

### 9.3 ホバー状態（Web）

**カード**:
- **背景**: グラデーション + 透明度アップ
- **シャドウ**: 1段階上げる
- **継続時間**: 200ms

## 11. ダークモード対応

### 11.1 自動切り替え

- システムの設定に応じて自動切り替え
- ユーザーが手動で切り替え可能

### 11.2 ダークモード固有の調整

**背景**:
- より暗い背景色（`#0F0E0C`）
- グラデーション調整（深い暖色系）

**テキスト**:
- より明るいテキスト色（`#F5F3F0`）
- コントラスト確保

**Glassmorphism**:
- より濃いガラス背景（透明度調整）
- ボーダーの透明度調整

## 12. アクセシビリティ

### 12.1 コントラスト比

- テキスト：最小 4.5:1（WCAG AA準拠）
- 大きなテキスト：最小 3:1

### 12.2 タッチターゲット

- 最小サイズ：44px × 44px
- ボタン間隔：最小 8px

### 12.3 カラーコントラスト

- グラデーション背景でも十分なコントラスト確保
- テキストは単色で統一

## 13. 実装ガイドライン

### 13.1 React Native での実装

**Glassmorphism効果**:
```typescript
import { BlurView } from 'expo-blur';

<BlurView intensity={80} style={styles.glassCard}>
  {/* Content */}
</BlurView>
```

**グラデーション**:
```typescript
import { LinearGradient } from 'expo-linear-gradient';

<LinearGradient
  colors={['#FF9A56', '#FFB347']}
  start={{ x: 0, y: 0 }}
  end={{ x: 1, y: 1 }}
  style={styles.gradient}
>
  {/* Content */}
</LinearGradient>
```

### 13.2 NativeWind での実装

**カスタムクラス定義**:
```css
@layer components {
  .glass-card {
    @apply rounded-2xl border border-white/20 shadow-lg;
    background: rgba(255, 255, 255, 0.8);
    backdrop-filter: blur(10px);
  }
  
  .gradient-primary {
    @apply bg-gradient-to-br from-orange-400 to-amber-300;
  }
}
```
